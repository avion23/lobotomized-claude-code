Broadcast a connection request to every Chrome browser with the extension installed and wait (up to 2 minutes) for the user to click 'Connect' in the one they want; they can name it on connect. Use when the user wants to pick the browser from inside Chrome; otherwise prefer select_browser with a known deviceId.
