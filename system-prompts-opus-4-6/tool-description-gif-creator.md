Manage GIF recording and export for browser automation sessions: start/stop recording browser actions, then export as an animated GIF with visual overlays (click indicators, action labels, progress bar, watermark). All operations are scoped to the tab's group. When starting recording, screenshot immediately after to capture the first frame; when stopping, screenshot immediately before to capture the last frame. For export, provide 'coordinate' to drag/drop onto a page element, or set 'download: true' to download the GIF.
