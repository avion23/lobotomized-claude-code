    <when_to_save>When you learn who is doing what, why, or by when. These states go stale quickly — when you notice one has, delete the memory and save a fresh one. Convert relative dates to absolute when saving (e.g., "Thursday" → "2026-03-05") so the memory stays interpretable later.</when_to_save>
